# Spring Weather Checker
A springboot project to check the weather using the free Geocoding and Weather API

## Goals
- Create a project that will showcase my ability to configure and use spring boot
- Utilize industry standard tools such as SonarQube to have secure and tested code
- Design a workflow on github in order to automate my development process
- Maintain security by using GPG encryption (ECC Standard)
- Containerize my project using Docker
