# Spring Weather Checker
A Spring Boot project to check the weather using the free Geocoding and Weather API

## Goals
- Create a project that will showcase my ability to configure and use Spring Boot
- Utilize industry standard tools such as SonarQube to have secure and tested code
- Design a workflow on github in order to automate my development process
- Maintain security by using GPG encryption (ECC Standard)
- Containerize my project using Docker
- Test code with SonarQube, JUnit, and Mockito

## Version 1.0
- [ ] Create a front end UI that will allow a user to input a city, hit search, and have the current weather appear below
- [ ] Create a back end that takes in the user input and makes API calls in order to find the weather using the free API

## Version 1.1
- [ ] Create a database that will store user search history

## Version 2.0
- [ ] Integrate google authentication for maintaining separate histories for separate users
